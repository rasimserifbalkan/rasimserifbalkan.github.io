Singleton Desing Pattern

Bu tararım deseninde tüm ama uygulamanın yaşam döngüsü boyunca sadece bir örnek üretilmesini sağlamaktır. Üretilmiş bir örnek bulunamadığında o örnek tekrardan oluşturulur.


Önemli bir konu


Singleton ile tekilleştirilecek olan sınıflar kendi içlerinde sadece method bazında işlem yapmalıdır.Herhangi bir data tutuyorsa bu data tüm kullanıcılar için ortak olmalıdır.Kullanıcı bazında bir data tutulacaksa bu sınıf singleton ile instance edilmemelidir.

Çok nadir kullanılan nesneler Singleton ile tekilleştirilmemelidir.


Şimdi kullanımına bakalım.


    public class Singleton
    {
        static Singleton instance;
      
        protected Singleton()
        {
        }
        public static Singleton Instance()
        {
            //Thread safe değildir.
            if (instance == null)
            {
                instance = new Singleton();
            }
            return instance;
        }
    }


Kullanımında nesneleri karşılaştırıyoruz.


    internal class Program
    {
        static void Main(string[] args)
        {
            var s1 = Singleton.Instance();
            var s2 = Singleton.Instance();
            if (s1 == s2)
            {
                Console.WriteLine("Aynı nesne örnekleri.");
            }
        }
    }


Şimdi gerçek hayattaki kullanımlarına bakalım.


Generic olarak singleton pattern kullanılabilir. Burada amacımız ger class içine singleton patterni yazmamaktır.


    public class SingletonRW<T>
    {
        private static T instance;
        public static T Instance
        {
            get => instance;
            set
            {
                instance = value;
            }
        }
    }


Şimdi bir BaseSingleton sınıfını oluşturacağız amacımı Singleton patterni kullanılarak tekilleştirilmiş sınıfları görebilmek.



    public class BaseSingleton
    {
        static BaseSingleton()
        {
            AllSingletons = new Dictionary<Type, object>();
        }
 
        /// <summary>
        /// Singleton örneklerini Dictionary içinde tutuyoruz.
        /// </summary>
        public static IDictionary<Type, object> AllSingletons { get; }
    }


Şimdi SingletonRW sınıfımı BaseSingleton sınıfından kalıtım alsın.


    public class SingletonRW<T> : BaseSingleton
    {
        private static T instance;
        public static T Instance
        {
            get => instance;
            set
            {
                instance = value;
                AllSingletons[typeof(T)] = value; // örneklenen nesneyi dictionary içerisine ekliyoruz.
            }
        }
    }


Artık nasıl kullanacağımıza bakabiliriz.Singleton için kullanacağımız MyObject isminde bir sınıf oluşturdum.


    public class MyObject
    {
        public void Run()
        {
            Console.WriteLine("Run");
        }
    }



    internal class Program
    {
        static void Main(string[] args)
        {
            var s1 = Singleton.Instance();
            var s2 = Singleton.Instance();
            if (s1 == s2)
            {
                Console.WriteLine("Aynı nesne örnekleri.");
            }
 
            // MyObject instance yoksa oluşturduk bu işlemi kod içerisinde yapabiliriz.
            var myObject = SingletonRW<MyObject>.Instance ?? (SingletonRW<MyObject>.Instance = new MyObject());
            myObject.Run();
        }
    }


Son hamle olarak bakalım arka planda hangi nesneler duruyor.
